# 예산 재배분 기획안

## 0. 개요

- 목적: insight_report(진단, '무슨 일이 있었나')의 **처방** — 예산으로 무엇을 할지 제안.
- **재원 원칙**: 조치 레버가 '예산'인 손실(광고비 과집행의 초과지출)만 재원으로 삼는다. 이상치·결측(데이터·트래킹 레버)은 데이터 수집·기록 오류라 예산 재배분으로 해결되지 않으므로 제외.
- **옮기는 금액 = 회수가능액**(실제 낭비한 광고비, 확실). **우선순위 크기 = 임팩트**(놓친 매출, 선형확장 추정).

## 1. 재원 요약 (어디서 얼마를 회수할 수 있나)

| 재원 채널 | 회수가능액 | 기회손실(임팩트) | 빈도 | 발생 |
|-----------|-----------|-----------------|------|------|
| 메타광고 | 3,194,281원 | 8,822,424원 | 1주 | W5 |

> 회수가능액(옮길 돈)은 실지출이라 확실하고, 기회손실(임팩트)은 '평소 효율이었다면' 가정이 섞인 추정치다.

## 2. 재배분안 (우선순위 순)

### [재배분안 #1] 우선순위 1순위  (임팩트 8,822,424원 × 빈도 1주 = 점수 8,822,424)

1. **문제 정의** — 메타광고 과집행: 회수 가능한 저효율 지출 발생(insight_report 손실 이슈 참조).
2. **근거 데이터** — 회수가능액 3,194,281원 / 재원채널 평소 ROAS 2.59 / 수혜채널 근거: ROAS 6.59 — 시장 벤치마크 상위25% 이상(증액 여력 검증)
3. **재배분안** — **메타광고 → 네이버광고**, 3,194,281원(테스트 상한), 점진 증액 후 재측정
4. **기대효과·한계** — 순증 매출 상한 약 12,772,685원 — 회수액을 평소 ROAS 2.59인 메타광고 대신 6.59인 네이버광고에 재투입 시 매출 순증 상한 (선형확장 가정).
  - 한계: 선형확장 가정 상한이라 한계수익 체감으로 실제는 하회 가능. 소액 테스트로 검증 후 확대.

### [재배분안 #2] 우선순위 2순위  (임팩트 8,822,424원 × 빈도 1주 = 점수 8,822,424)

1. **문제 정의** — 메타광고 과집행: 회수 가능한 저효율 지출 발생(insight_report 손실 이슈 참조).
2. **근거 데이터** — 회수가능액 3,194,281원 / 재원채널 평소 ROAS 2.59 / 수혜채널 근거: 광고비 직접지출 0(무료 아님 — 과거 유료광고 낙수효과·SEO 누적 결과), 매출 104,238,814(전체 23.71%), CVR 3.11%
3. **재배분안** — **메타광고 → 오가닉**, 3,194,281원 범위 내 SEO·콘텐츠 투자(광고비 직접 배정 아님)
4. **기대효과·한계** — 오가닉 광고비 직접지출 0(무료 아님 — 과거 유료광고 낙수효과·SEO 누적 결과), 매출 104,238,814(전체 23.71%), CVR 3.11% — SEO·콘텐츠 투자로 중장기 유입 성장(즉시 ROAS 미정).
  - 한계: 오가닉은 ROAS 측정불가라 원 단위 기대효과를 추정하지 않음(정성). 효과 발현에 시차 존재.

> **재원 공유 주의**: 위 안들은 같은 재원(과집행 회수분)을 공유한다 — 돈은 한 번만 쓴다. 1순위를 우선 집행하고, 성과를 확인한 뒤 잔여·후속분을 다음 순위로 순차 배분한다.

## 3. 우선순위 판단 기준 (설계 근거)

재배분안을 어떤 순서로 실행할지는 아래 기준으로 결정한다. 임의 판단이 아니라 데이터로 정렬한다.

1. **1차 정렬 = 임팩트 × 빈도.** 임팩트(놓친 매출)만 보면 '1주짜리 대형 사고'와 '매주 조금씩 새는 구멍'을 구분 못 한다. 빈도(발생 주차 수)를 곱하면 **반복되는 구조적 낭비**가 위로 올라온다 — 예산 재배분의 진짜 타깃.
2. **2차 정렬(동점) = 수혜처 즉효성.** 같은 재원을 공유하면 임팩트·빈도가 같아 점수가 동일해진다. 이때 **즉시 ROAS가 실현되는 광고 증액**을 효과가 지연되는 전략 투자(SEO·콘텐츠)보다 우선한다.
3. **옮기는 금액 = 회수가능액(≠임팩트).** 실제로 회수 가능한 초과지출만 옮긴다. 임팩트(추정 손실)를 재배분액으로 쓰면 존재하지 않는 돈을 옮기는 허수 기획이 된다.
4. **수혜처 선정 = 절대 ROI가 아니라 시장 벤치마크 정성 등급.** 절대 ROI 1위 채널은 이미 예산이 포화됐을 수 있다(한계수익 체감). '자기 시장 대비 우수(상위25%↑)'라 증액 여력이 검증된 채널을 고른다.

**한계(정직한 고지)**: 광고 증액 기대효과는 선형확장 가정 상한이라 실제는 하회할 수 있고, 오가닉 기대효과는 정성으로만 제시한다. 모든 안은 소액 테스트 → 성과 재측정 → 확대의 검증 절차를 전제한다.

## 4. 방법론·재현성

> 재원·수혜·우선순위는 전부 Python(calculate.py·detect_issues.py·reallocate.py)이 계산하고, 해석·문구는 규칙 기반 템플릿으로 채운다. 새 성과 CSV를 넣으면 동일 구조로 자동 재생성된다 (`python3 src/reallocate.py [입력CSV] [출력MD]`).
